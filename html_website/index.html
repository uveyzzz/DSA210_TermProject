Methodology

How the Analysis Works

A systematic approach to data processing, merging, and statistical analysis.

1

Data Collection & Export

Personal data exported from Meta (Instagram) and Apple Health. Instagram data includes JSON files with liked posts metadata. Apple Watch data exported as CSV with detailed workout metrics.

2

Fitness Content Classification

A keyword-based classifier identifies fitness-related Instagram interactions using 50+ keywords in both English and Turkish, including hashtags (#fitness, #gym, #workout) and account handle patterns.

3

Daily Aggregation & Merging

Both datasets aggregated to daily level using date as the key. Final merged dataset contains 370 days with metrics like total_like, fitness_like, total_workout_minutes, calories_burned, and workout_done binary indicator.

4

Statistical & ML Analysis

Pearson correlation computed between Instagram metrics and workout variables. Welch's t-test and Mann-Whitney U tests compare workout metrics. Additionally, machine learning models (Logistic Regression, Random Forest, Ridge Regression) predict workout occurrence and duration from Instagram engagement patterns.

Conclusions

Key Takeaways

What we learned from this personal data exploration.

The data suggests a weak positive association between fitness content engagement and workout behavior. Machine learning models show moderate predictive power using Instagram patterns.

üîç
Finding 1
Correlation ‚â† Causation
While days with more fitness content engagement show higher workout probability, this could be reverse causality (workout days ‚Üí more fitness interest).
üìä
Finding 2
ML Shows Promise
Random Forest achieves AUC ~0.72 for workout prediction, suggesting Instagram patterns contain useful signals for predicting exercise behavior.
üí°
Finding 3
Behavioral Complexity
Exercise motivation is multifactorial. Social media is just one potential influence among many lifestyle factors.
üìà
Finding 4
Lag Features Matter
Previous day's workout status and rolling averages were important features, highlighting the importance of temporal patterns in behavior prediction.
